# sysuManager
运行该该网页

step1:下载tomcat服务器，以及mysql数据库

step2:将WebRoot文件夹放入服务器部署环境中（如为tomcat，将WebRoot复制到webapps文件夹下）

step3:数据库部署(默认用户：root；密码：1234)：将sql/sysu.sql文件在mysql数据库中还原

step4:在浏览器输入“http://localhost:8080/WebRoot/”，即可进入网页
